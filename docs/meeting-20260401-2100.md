## 2. LINE相談の導線

- トップページのヘッダーにLINE相談は**不要**（いきなり相談にはならない）
- 追従バナー（スクロールくっつき）は**スパムっぽいのでNG**
- 求人詳細ページやコラム記事内に**文脈に沿って配置**する方針
- FAQ下のCTAは2つあると迷う → **LINE無料相談のみ**に統一
- FAQで解決しない場合の受け皿として「不明点があればここに投げてください」という**丁寧な誘導**にする

## 3. FAQ・応募の流れ

- 応募の流れ、契約形態は**FAQ内に統合**する（別枠にしない）
- 加えて「マイリモとは」ページ内でも**文脈に合わせて盛り込む**（一箇所限定にしない）
- 初めてリモートワークする人の不安に寄り添う内容が必要（面談の有無、オンラインか訪問かなど）

## 4. コラム・ブログの方針

- コラムの**カテゴリ整理・導線改善**は記事が溜まってからでOK（今は優先度低）
- 関連記事の表示自体は賛成（サイト滞在→ファン化→求人応募への導線）
- **福本さんがコラム執筆**する方向。加藤さんがテーマ案をまとめ、それに沿って福本さんが書く
- ターゲットに合わせた内容:
  - フリーランス・業務委託で掛け持ちしている人
  - フルリモート初挑戦の人
  - 働く上でのマインドセット、契約形態の違い、職種別の働き方

## 5.「マイリモとは」ページの構成

- ターゲット: テック系・ハイスキル人材寄り（レバテック等のイメージ）
- ただしテレアポ人材も流入するため、**どちらにフォーカスするかでメッセージが変わる**
- 方向性: 在宅ワークでの不満・不安を**代弁**し、マイリモなら安心して始められる理由を示す
- 縁結びとしての理念・行動指針をインプットし、スタート時点で利他的に振る舞える状態を目指す
